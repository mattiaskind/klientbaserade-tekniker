... det här p-elementet

CSS-koden anger att färgen ska appliceras på både h2- och p-elementen. Selektorerna är separerade med kommatecken. 

CSS
h2, p {
    color: blueviolet;
}

HTML
<h2>Den här rubriken har samma stilsättning som...</h2>
<p>... det här p-elementet</p>

Det går också att ange att ett HTML-element ska ha flera klasser genom att separera dem med ett blanksteg i class-attributet. Exempelvis texten nedan som är färgsatt och har fått en ram med hjälp av två separata klass-selektorer:

Blå text med ram runt 
CSS
.my-color {
    color: blue;
}
.my-border {
    border: 1px solid black;
}

HTML
<span class="my-color my-border">Blå text med ram runt</span>
