CSS

Cascading Style Sheet (CSS) är ett språk som används för att stilsätta HTML-dokument. Med CSS beskriver man hur de olika HTML-elementen ska visas och vilka egenskaper de ska ha, exempelvis vilken färg det ska vara på texten. Se vad CSS gör med den här sidan genom att klicka på knappen nedan.

Hur funkar det?

Om vi utgår från att vi vill ändra bakgrundsfärgen på body-elementet i ett HTML-dokument till ljusblå kan vi göra på följande sätt: 

body {
  background-color: lightblue;
}

I exemplet ovan har vi angett body för att stilsätta det elementet. Inom måsvingarna anger vi vilken egenskap vi vill stilsätta, vi vill alltså ändra bakgrundsfärgen, sedan anger vi vilket värde vi vill ändra till. Tillsammans utgör ovanstående en regeluppsättning. Olika egenskaper kan ges olika värden, vi kan till exempel även ändra font-storleken genom att lägga till följande: 

body {
  background-color: lightblue;
  font-size: 30px;
}
Se exemplet

Varje regeluppsättning måste omslutas av måsvingar ({ }). Varje egenskap och dess värde måste separeras med kolon (:) och varje uppgift, dvs. egenskap/värde måste avslutas med ett semikolon (;). Det är viktigt att hålla sig till reglerna för att CSS-koden ska fungera.

Stilsätta ett dokument

Var ska man skriva sin CSS-kod? Det finns några olika metoder att använda för att lägga in CSS-kod.

Extern CSS

Den bästa praxisen är normalt att använda en extern fil. Då kan du skriva all din css-kod separat. Det gör det möjligt att använda samma css-fil i flera dokument på din webbplats. För att länka en extern css-fil i ditt HTML-dokument lägger du in nedanstående kod inom <head>-taggen.

<link rel="stylesheet" href="style.css" />
Notera att href-taggen anger platsen för din css-fil. I exemplet ovan heter filen style.css. Om du är osäker kan du läsa lite om sökvägar under kategorin HTML.

Så här kan det se ut i ett dokument med en länkad css-fil:

<!DOCTYPE html>
<html lang="sv">
  <head>
    <meta charset="UTF-8" />
    <title>Sidan titel</title>
    <link rel="stylesheet" href="style.css" />
  </head>
...

Intern CSS

Med intern silsättning skrivs css-koden i samma dokument som HTML-koden. CSS-koden omsluts då av style-taggar. 

<!DOCTYPE html>
<html lang="sv">
  <head>
    <meta charset="UTF-8" />
    <title>Sidan titel</title>
    <link rel="stylesheet" href="style.css" />
    <style>
      body {
        background-color: blue;
      }
    </style>
  </head>
...

Inline CSS

Slutligen går det att applicera stilsättning direkt på ett specifikt element i HTML-koden. Då används attributet style. I exemplet nedanför appliceras stilsättning direkt på ett p-element. 

<!DOCTYPE html>
<html lang="sv">
  <head>
    <meta charset="UTF-8" />
    <title>Sidan titel</title>
  </head>
  <body>

  <p style="color: blue;">Den här texten blir blå.</p>
  <p>Men det blir inte den här.</p>

  <body>
</html>

Klasser och ID:n

Klasser

I det inledande exemplet applicerade vi en bakgrundsfärg på body-elementet. För att välja ett element skrev vi body följt av måsvingar och applicerade sedan de egenskaper vi önskade på elementet. På samma sätt kan vi göra med andra element, vi kan exempelvis ändra färgen på alla rubriker av typen <h3> genom att skriva så här. 

h3 {
  color: green;
}

Med ovanstående kod anger vi alltså att samtliga element av typen <h3> ska ha färgen grön. Om vi enbart vill ändra färgen på en rubrik, inte alla, kan vi använda oss av klasser eller ID:n. I exemplet nedanför vill vi ändra färg på den första rubriken och det första stycket, men inget annat, så vi skapar en klass. 

.green {
  color: green;
}
Notera punkten (.) före klassnamnet.

Klassen .green kan vi sedan använda på specifika element 

<body>
  <h3 class="green">Rubrik 1</h3>
  <p class="green">Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>

  <h3>Rubrik 2</h3>
  <p>Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>
</body>
ID

Istället för klasser går det också att använda sig av ID:n. Det fungerar på ungefär samma sätt. Istället för att ange en punkt (.) före namnet på klassen anges (#). Till skilland från klasser ska ID:n vara unika. Det ska alltså bara finnas ett element som har ett specifikt ID per HTML-dokument. 

#menu {
  color: green;
}

Prioriteringsregler

Vad händer när ett element stilsätts flera gånger? Vad avgör vilken stilsättning som kommer att appliceras? Vi tar ett exempel med följande CSS-kod: 

h2 {
  color: green;
}
.blue {
  color: blue;
}
#my-heading {
  color: orange;
}

Och följande HTML-kod.

<body>
  <h2 class="blue" id="my-heading">Min rubrik</h2>  
</body>
Se resultatet av exemplet ovan

Så här fungerar det, ju högre siffra desto högre relevans:

  1. typ-selektorer, som h2 i exemplet
  2. klass-selektorer som .blue i exemplet
  3. ID-selektorer som #my-heading i exemplet. Därför visas texten som orange (se länk till exemplet ovan).
  4. Inline-stil - om man applicerar stilsättning direkt på elementet i HTML-koden skriver det över extern stilsättning.
!important

Det finns möjlighet att kringgå prioriteringsreglerna i CSS genom att ange !important. När important används skriver regeln över alla andra regler. Det är normalt inte att rekommendera eftersom man därigenom frångår den naturliga ordningen, det blir svårare att felsöka. Man bör istället använda sig av den inbyggda ordningen så långt som möjligt eftersom koden blir mer förutsägbar.

I CSS kan egenskaper ärvas

Om ett element inte stilsätts ärver det i många fall egenskaperna från sin förälder. 

body {
  color: green;
}

All text som ligger inom <body>-elementet, oavsett det är text inom <p>, <h1>, <h2>, <h3>, <h4> osv. kommer att bli grön om inget annat anges för just det elementet.

Se ett exempel

Float

Med egenskapen float kan ett element placeras på vänster eller höger sida av sitt föräldraelement. Float-egenskapen gör att elementet tas bort från det normala flödet men det ingår ändå som en del av dokumentet och text samt andra inline-element kan flöda runt elementet.

Med float förflyttas elementet åt vänster eller höger till dess att det nuddar kanten av antingen boxen det placerats i eller ett annat element som har egenskapen float.

Se ett exempel 
.container {
  float: left;
}

Positionering

Med hjälp av egenskapen position är det möjligt att använda CSS för att positionera HTML-element. Det finns fem olika värden som kan anges:

  • static - elementet positioneras i enlighet dokumentets ordinarie flöde. Detta är standardvärdet.
    • Se hur static ser ut
  • relative - elementet följer det vanliga flödet men kan förflyttas med hjälp av värdena, top, right, bottom och left
    • Ett exempel som visar relative
  • absolute - elementet tas bort från det vanliga flödet och det utrymme som normalt beräknas för elementet tas bort. Elementets position beräknas utifrån föräldraelementet.
    • Ett exempel som visar absolute
  • fixed - elementet tas bort från det vanliga flödet och dess position beräknas utifrån webbläsarfönstret
    • Ett exempel som visar fixed
  • sticky - elementets position avgörs av hur användaren scrollat webbläsarfönstret. När användaren scrollar förbi sticky-elementet uppför det sig som om det vore "fixed", till dess att föräldraelementet passerats. Det är knepigt att förklara hur det funkar, klicka på länken nedan för ett exempel.
    • Ett exempel som visar sticky
