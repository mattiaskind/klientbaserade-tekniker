CSS

Cascading Style Sheet (CSS) är ett språk som används för att stilsätta HTML-dokument. Med CSS beskriver man hur de olika HTML-elementen ska visas. Se vad CSS gör med den här sidan genom att klicka på knappen nedan.

Hur funkar det?

Om vi utgår från att vi vill ändra bakgrundsfärgen på body-elementet i ett HTML-dokument till mörkblå kan vi göra på följande sätt: 

body {
  background-color: darkblue;
}

I exemplet ovan har vi angett body för att stilsätta det elementet. Inom måsvingarna anger vi vilken egenskap vi vill stilsätta, vi vill alltså ändra bakgrundsfärgen, sedan anger vi vilket värde vi vill ändra till. Tillsammans utgör ovanstående en regeluppsättning. Olika egenskaper kan ges olika värden, vi kan till exempel även ändra font-storleken genom att lägga till följande: 

body {
  background-color: darkblue;
  font-size: 20px;
}

Varje regeluppsättning måste omslutas av måsvingar. Varje egenskap och dess värde måste separeras med kolon (:) och varje uppgift, dvs. egenskap/värde måste avslutas med ett semikolon (;). Det är viktigt att hålla sig till reglerna för att CSS-koden ska fungera.

Stilsätta ett dokument

Var ska man skriva sin CSS-kod? Det finns några olika metoder att använda för att lägga in CSS-kod, vi ska gå igenom dessa nu.

Extern CSS

Den bästa praxisen är normalt att använda en extern fil. Då kan du skriva all din css-kod separat. Det gör det möjligt att använda samma css-fil i flera dokument på din webbplats. För att länka en extern css-fil i ditt HTML-dokument lägger du in nedanstående kod inom <head>-taggen.

<link rel="stylesheet" href="style.css" />
Notera att href-taggen anger platsen för din css-fil. I exemplet ovan heter filen style.css. Om du är osäker kan du läsa lite om sökvägar under kategorin HTML här på sidan.

Så här kan det se ut i ett dokument med en länkad css-fil:

<!DOCTYPE html>
<html lang="sv">
  <head>
    <meta charset="UTF-8" />
    <title>Sidan titel</title>
    <link rel="stylesheet" href="style.css" />
  </head>
...

Intern CSS

Med intern silsättning skrivs css-koden i samma dokument som HTML-koden. CSS-koden omsluts då av style-taggar. 

<!DOCTYPE html>
<html lang="sv">
  <head>
    <meta charset="UTF-8" />
    <title>Sidan titel</title>
    <link rel="stylesheet" href="style.css" />
    <style>
      body {
        background-color: blue;
      }
    </style>
  </head>
...

Inline CSS

Slutligen går det att applicera stilsättning direkt på ett specifikt element i HTML-koden. Då används attributet style="". I exemplet nedanför appliceras stilsättning direkt på ett p-element. 

<!DOCTYPE html>
<html lang="sv">
  <head>
    <meta charset="UTF-8" />
    <title>Sidan titel</title>
  </head>
  <body>

  <p style="color: blue;">Den här texten blir blå.</p>
  <p>Men det blir inte den här.</p>

  <body>
</html>

Klasser och ID:n

Klasser

I det inledande exemplet applicerade vi en bakgrundsfärg på body-elementet. För att välja ett element skrev vi body följt av måsvingar och applicerade sedan de egenskaper vi önskade på elementet. På samma sätt kan vi göra med andra element, vi kan exempelvis ändra färgen på alla rubriker av typen <h3> genom att skriva så här. 

h3 {
  color: green;
}

Med ovanstående kod anger vi generellt att samtliga element av typen <h3> ska ha färgen grön. Om vi enbart vill ändra färgen på en rubrik, inte alla, kan vi använda oss av klasser eller ID:n. I exemplet nedanför vill vi ändra färg på den första rubriken och det första stycket, men inget annat, så vi skapar en klass. 

.green {
  color: green;
}
Notera punkten (.) före klassnamnet.

Klassen .green kan vi sedan använda på specifika element 

<body>
  <h3 class="green">Rubrik 1</h3>
  <p class="green">Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>

  <h3>Rubrik 2</h3>
  <p>Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>
</body>
ID

Istället för klasser går det också att använda sig av ID. Det fungerar på ungefär samma sätt. Istället för att ange en punkt (.) före namnet på klassen anges tecknet (#). Till skilland från klasser är tanken med ID:n att dessa ska vara unika. Det ska alltså bara finnas ett element som har ett specifikt ID per HTML-dokument. 

#menu {
  color: green;
}

Block och inline

I CSS omgärdas allting av boxar. Det finns generellt två varianter av boxar: block-boxar och inline-boxar. Egenskaperna för respektive variant avgör hur boxen anpassar sig till HTML-dokumentets flöde.

Block-element skapar alltid ett radbyte medan inline-element inte gör det. Vi ska titta närmare på två sådana varianter. Elementet <div> är av typen block och elementet <span> är av typen inline. 

<div>Efter det här innehållet följer ett radbyte.</div>
<p>Lorem ipsum <span class="blue">inget radbyte</span>amet elit.</p>
